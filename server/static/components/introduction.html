Talon Noise Project

Introduction

Thank you for contributing to the Talon Noise Project. We are collecting recordings to train a deep neural network for Talon. It will recognize sounds such as clicks, pops, and hisses and allow controlling a computer using only a microphone.

Contributing

Note that this will take approximately 20 to 30 minutes to complete.

  1. Pick a sound to record. You can start with click, which is the first sound.
  2. Read the sound's instructions and listen to the example recording.
  3. When you think you are ready to record the sound, press the round Record button.
  4. After pressing Record, wait for about one second, then make the sound 5-10 times to your comfort, pausing about one second between each time you make the sound.
  5. When you are done recording, press the square Stop button. The sequence will look like this:

    press Record → wait → make sound → wait → make sound → ... → press Stop

  6. You can then select another sound from the list and repeat the process.
  7. If you finish recording all of the sounds, feel free to go back and record more samples at any time.
  8. These sounds will be used to train a neural network. More recordings as well as different voices, microphones, and environments will help the training.

More information about the sounds can be found here.

If you have questions or comments, join the Talon Slack and ask in the #noise channel.
